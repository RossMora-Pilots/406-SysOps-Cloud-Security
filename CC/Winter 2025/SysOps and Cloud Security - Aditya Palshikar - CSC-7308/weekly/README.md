## Summary Format

Each weekly summary follows this structure:

1. **Session Info** — date, duration, lab reference
2. **Topics Covered** — what was taught
3. **Tools / Platforms** — hands-on technology
4. **Key Concepts** — the ideas that matter
5. **Lab Deliverable** — what was submitted
6. **Reflection** — personal synthesis and takeaways
7. **Connections** — how it links to other weeks or courses
