Based on course progression, the final assessment is expected to integrate:

1. **Comprehensive cloud security review** — IaaS/PaaS/SaaS posture assessment
2. **Multi-layer defense demonstration** — NGFW + SIEM + endpoint + cloud controls working together

4. **Incident response exercise** — full kill-chain scenario from initial detection through containment and lessons learned
