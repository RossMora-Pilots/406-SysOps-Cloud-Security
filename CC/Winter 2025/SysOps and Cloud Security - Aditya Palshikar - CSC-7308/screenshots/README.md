## Naming Convention

- **Weekly lab:** `wkNN_<topic>_<index>.png`
  - Example: `wk01_firewall_logs_1.png`, `wk04_cyber_kill_chain_1.png`
- **General evidence:** `ScreenshotN_<short-desc>.png`

## Rules

- Lowercase filenames; use underscores or dashes, no spaces.
- Redact PII before capture (student names, emails, internal hostnames, tokens).
- Reference each image from [`../EVIDENCE_INDEX.md`](../EVIDENCE_INDEX.md) with a caption.
- PNG preferred for UI screenshots; JPG acceptable for photos.
