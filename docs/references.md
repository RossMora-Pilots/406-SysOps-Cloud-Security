# References — Annotated Bibliography

Curated references for further study of SysOps and Cloud Security topics.

## Palo Alto Networks (Vendor Documentation)

- **[PAN-OS Administrator's Guide](https://docs.paloaltonetworks.com/pan-os)** — Canonical reference for NGFW configuration.
- **[AutoFocus Product Page](https://www.paloaltonetworks.com/cortex/autofocus)** — Threat intelligence platform.
- **[WildFire Documentation](https://docs.paloaltonetworks.com/wildfire)** — Cloud malware analysis service.
- **[Prisma Cloud](https://www.paloaltonetworks.com/prisma/cloud)** — Cloud security posture management.
- **[Cortex XDR](https://www.paloaltonetworks.com/cortex/xdr)** — Extended Detection and Response.

## Wazuh (Open-Source SIEM)

- **[Wazuh Documentation](https://documentation.wazuh.com/)** — Full reference for deployment and operation.
- **[Wazuh Ruleset](https://github.com/wazuh/wazuh-ruleset)** — Open-source detection rules repository.
- **[Wazuh Blog](https://wazuh.com/blog/)** — Threat hunting and detection engineering articles.

## Frameworks & Standards

- **[Lockheed Martin Cyber Kill Chain](https://www.lockheedmartin.com/en-us/capabilities/cyber/cyber-kill-chain.html)** — Seven-stage attack framework used in Week 4.
- **[MITRE ATT&CK](https://attack.mitre.org/)** — Tactics and techniques catalog (complements Kill Chain).
- **[NIST SP 800-145](https://csrc.nist.gov/publications/detail/sp/800-145/final)** — NIST Definition of Cloud Computing.
- **[NIST SP 800-190](https://csrc.nist.gov/publications/detail/sp/800-190/final)** — Application Container Security Guide.
- **[NIST SP 800-83](https://csrc.nist.gov/publications/detail/sp/800-83/rev-1/final)** — Malware Incident Prevention and Handling.
- **[NIST SP 800-181](https://www.nist.gov/itl/applied-cybersecurity/nice/nice-framework-resource-center)** — NICE Cybersecurity Workforce Framework.
- **[CSA Cloud Controls Matrix](https://cloudsecurityalliance.org/research/cloud-controls-matrix)** — Cloud security control mapping.
- **[CIS Benchmarks](https://www.cisecurity.org/cis-benchmarks)** — Configuration baselines for OSes, containers, cloud.

## Cloud Provider Security

- **[AWS Security Best Practices](https://aws.amazon.com/security/)** — Amazon Web Services.
- **[Microsoft Security Documentation](https://learn.microsoft.com/en-us/security/)** — Azure, Defender, Entra ID.
- **[Google Cloud Security](https://cloud.google.com/security)** — GCP security center.

## Container & Kubernetes

- **[Kubernetes Pod Security Standards](https://kubernetes.io/docs/concepts/security/pod-security-standards/)** — Privileged/baseline/restricted profiles.
- **[CIS Kubernetes Benchmark](https://www.cisecurity.org/benchmark/kubernetes)** — Cluster hardening guide.
- **[Docker Security](https://docs.docker.com/engine/security/)** — Container runtime security.

## Rust Programming

- **[The Rust Programming Language](https://doc.rust-lang.org/book/)** — The Rust Book.
- **[Tokio Async Runtime](https://tokio.rs/)** — Used in the ping-sweep project.
- **[Rust by Example](https://doc.rust-lang.org/rust-by-example/)** — Worked examples.

## Adjacent Academic Reading

- Hutchins, Cloppert, Amin. *Intelligence-Driven Computer Network Defense Informed by Analysis of Adversary Campaigns and Intrusion Kill Chains*. Lockheed Martin, 2011. (The original Cyber Kill Chain paper.)
- Anderson, Ross. *Security Engineering* (3rd ed.). Wiley, 2020. (Architectural thinking for security systems.)
- Bejtlich, Richard. *The Practice of Network Security Monitoring*. No Starch Press, 2013. (SOC analyst fundamentals.)
